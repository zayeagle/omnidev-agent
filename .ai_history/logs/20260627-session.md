## Turn — 2026-06-27 — fix install.ps1 go build failed

### User
修复一下（`install.ps1` 执行 `go build failed`，终端显示 link 帮助信息）

### Assistant
**根因**：`buildTime` 使用 `yyyy-MM-dd HH:mm:ss` 格式，中间有空格。PowerShell 传给 `go build -ldflags` 时，`-X main.buildTime=2026-06-27 12:00:00` 被拆成多个参数，linker 解析失败并打印 usage。

**修复**：与 `Makefile` 一致，对含空格的 buildTime 值加单引号：

```powershell
go build -ldflags "-X main.appVersion=$version -X 'main.buildTime=$buildTime'" ...
```

**修改文件**：
- `scripts/install.ps1`
- `build/build.ps1`

**验证**：`powershell -File .\scripts\install.ps1` 成功，输出 `omnidev-agent v0.0.1  built 2026-06-27 13:15:55`。
